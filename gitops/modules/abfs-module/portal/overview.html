Horizon Module

Android Build File System (ABFS)

ABFS provides high-performance, distributed, and cached filesystem capabilities for scaling large-scale AOSP (Android Open Source Project) compilations and builds natively inside your Horizon GKE clusters.

KCC Infrastructure

Managing database states with Cloud Spanner, licensing payload storage in Cloud Storage (GCS), and secrets inside Secret Manager using native Config Connector custom resources.

Performance Caching

Utilizes the custom casfs kernel module compiled on Ubuntu Containerd nodes. Provides high throughput file replication and indexing backing the build pipeline.

Continuous Integration

Includes full integration with Argo Workflows for automated code seeding, verification runs, and workspace provisioning directly inside isolated module namespaces.
